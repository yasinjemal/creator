# CreatorFlow

CreatorFlow is a content generation platform for multi-platform social media management, optimization, and scheduling.

This repository contains:
- frontend/ — Nuxt 3 frontend (Vue.js)
- backend/ — Flask API server (Python)
- infra/ — docker-compose and deployment configs
- docs/ — integration docs and strategy notes
- scripts/ — helper scripts

Getting started (development)
1. Backend: create a Python virtualenv and install dependencies from `backend/requirements.txt`.
2. Frontend: run `npm install` inside `frontend/` and `npm run dev` (Nuxt).

See `docs/` for API, architecture, and scaling notes.
